离散状态空间模型生成器

项目概述

本MATLAB项目用于将连续时间传递函数转换为离散时间状态空间模型，包含三个模块化文件，便于系统分析与控制设计。

文件结构

1. generateTransferFunction.m

功能：生成连续时间传递函数

输入参数：
• num - 传递函数分子系数向量（可选）

• den - 传递函数分母系数向量（可选）

输出参数：
• G - 连续时间传递函数对象

默认系统：
• 二阶系统：G(s) = 1/(s² + 2.8s + 4)

• 自然频率 ω = 2 rad/s

• 阻尼比 ζ = 0.7

生成图形：
• 阶跃响应图

• 波特图

2. convertToDiscreteSS.m

功能：将连续传递函数转换为离散状态空间模型

输入参数：
• G - 连续传递函数对象

• Ts - 采样时间（默认：0.1秒）

• method - 离散化方法（默认：'zoh'）

输出参数：
• Ad, Bd, Cd, Dd - 离散状态空间矩阵

• sysd - 离散状态空间系统对象

支持的离散化方法：
• 'zoh' - 零阶保持器

• 'foh' - 一阶保持器

• 'tustin' - 双线性变换（Tustin方法）

生成图形：
• 离散系统阶跃响应

• 零极点图

3. main.m

功能：主程序，集成完整工作流程

交互功能：
1. 选择默认或自定义传递函数
2. 设置离散化参数
3. 执行系统转换
4. 比较连续与离散系统
5. 保存结果

生成图形：
• 连续与离散系统比较

• 离散化误差分析

使用方法

快速开始

% 运行主程序
main


手动调用函数

% 生成默认传递函数
G = generateTransferFunction();

% 转换为离散状态空间
[Ad, Bd, Cd, Dd, sysd] = convertToDiscreteSS(G, 0.1, 'zoh');


自定义系统

% 定义三阶系统
num = [1];
num_den = [1, 5, 10, 10];
G = generateTransferFunction(num, den);

系统要求

• MATLAB R2018b 或更高版本

• Control System Toolbox

注意事项

1. 离散化可能导致混叠效应，需满足香农采样定理
2. 建议采样频率 ≥ 10×系统带宽
3. 系统必须为真分式（分子阶数 ≤ 分母阶数）
